# Aakin & Palak's Wedding Website

This is the official wedding website for Aakin and Palak.

## Features
- **Event Timeline**: Detailed schedule of Haldi, Sangeet, and Ceremony.
- **RSVP System**: Guests can submit their attendance (connects to Google Sheets).
- **Gallery**: Integration for guests to upload photos.
- **Travel Info**: Hotel block details and local attractions.

## Technologies
- React
- TypeScript
- Tailwind CSS (via CDN)
- Vite

## Deployment
This project is configured to deploy automatically to **GitHub Pages** using GitHub Actions.
